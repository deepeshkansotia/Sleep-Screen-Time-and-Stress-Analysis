# Smartphone Usage, Sleep & Stress Analysis

## Project Overview
This project analyzes how smartphone usage and lifestyle habits impact sleep quality and stress levels. The analysis was conducted using Python for data cleaning and exploratory data analysis (EDA), PostgreSQL for analytical querying, and visualization techniques to uncover meaningful insights.

The goal of the project is to understand how factors such as screen time, caffeine intake, notifications, and physical activity influence sleep duration and stress levels.

---

## Dataset
The dataset contains behavioral and lifestyle data for users, including:

- Age
- Gender
- Occupation
- Daily screen time
- Sleep duration
- Sleep quality score
- Stress level
- Physical activity minutes
- Caffeine intake
- Notifications received per day
- Phone usage before sleep

---

## Project Workflow

### 1. Data Preprocessing (Python - Pandas)
- Handling missing values
- Removing duplicates
- Outlier detection
- Feature engineering (age groups, activity levels, notification levels)

### 2. Exploratory Data Analysis (Python)
Key visualizations include:

- Correlation heatmap
- Screen time vs stress regression analysis
- Sleep duration vs sleep quality scatter plots
- Stress distribution analysis
- Bubble plot showing screen time, sleep, stress, and activity
- Lifestyle risk map
- Radar chart for lifestyle profile

### 3. SQL Analysis (PostgreSQL)
SQL was used to extract deeper insights from the dataset, including:

- Stress levels by occupation
- Sleep duration across age groups
- Impact of physical activity on stress
- Effect of caffeine consumption on sleep
- Relationship between notifications and stress
- Identification of high screen-time users


## Key Insights

Some key insights discovered during analysis include:

- Higher screen time is associated with increased stress levels.
- Users with higher physical activity tend to report lower stress.
- Increased phone usage before sleep negatively affects sleep quality.
- Caffeine intake is correlated with reduced sleep duration.
